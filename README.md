# Glassic UI

A production-ready glassmorphic component library built on React, Radix UI, and Tailwind CSS.

## Features

- 5-level glass elevation system with adaptive blur, opacity, and border glow
- 15+ accessible, theme-adaptive components
- Dark & Light mode with full token support
- Glassmorphic Chart components (Area, Bar, Line, Pie)
- Copy-paste ready — each component is self-contained

## Tech Stack

- React 18+
- TypeScript
- Tailwind CSS
- Radix UI
- Framer Motion
- Recharts

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Install dependencies
npm install

# Start development server
npm run dev
```
