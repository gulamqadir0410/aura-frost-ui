
# Glassic UI — Documentation & Showcase Site

## Overview
A **glassmorphism-inspired component library** documentation and showcase site featuring live interactive demos, copyable code snippets, and comprehensive docs for each component.

---

## 1. Design System Foundation
- **Glass design token system** — 5 glass elevation levels with backdrop-blur, opacity, border glow, and shadow utilities
- Tailwind utility classes (`glass-1` through `glass-4`, `glass-float`)
- **Light & dark mode** support with adaptive glass opacity tokens
- Subtle motion effects (hover lift, focus glow, entrance fade)

## 2. Site Layout
- **Collapsible sidebar** with grouped navigation
- **Top header** with logo, theme toggle, and GitHub link
- **Main content area** with component docs, live previews, and code blocks
- Responsive design — sidebar collapses on mobile

## 3. Component Pages (15+ components)
Each component page includes live interactive demo, variants showcase, props/API table, copyable code snippets, and accessibility notes.

## 4. Example Templates
- **Crypto Price Tracker** (`/examples/crypto-tracker`) — full production-level landing page demo using exclusively Glassic UI components
